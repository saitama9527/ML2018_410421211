#### 遇到的問題 #
公式內矩陣轉置又轉置，寫的時候很容易搞混，爾且numpy轉置一維矩陣又有些麻煩，花了很多時間才弄懂。再來就是在反推圖片陣列的時候都是浮點數，而在轉整數時因為uint8的設定只能0~255，所以有些四捨五入後變成256的就被轉成0，導致圖片出現很多黑點，最後用一個判斷式調整數值避免爆表。
